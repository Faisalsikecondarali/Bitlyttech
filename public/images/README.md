# Images Directory

This directory contains all image assets used in the application.

## Structure
- `icons/` - Small icon images and logos
- `backgrounds/` - Background images and patterns
- `illustrations/` - Custom illustrations and graphics
- `screenshots/` - Application screenshots (if needed)

## Guidelines
- Use descriptive filenames (e.g., `hero-banner.jpg` instead of `img1.jpg`)
- Optimize images for web (use WebP format when possible)
- Keep file sizes reasonable for performance
- Use appropriate resolutions for different screen sizes
